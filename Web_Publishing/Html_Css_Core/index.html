스타일 우선 순위

      인라인 스타일 > 선택자에 id가 많음 > 선택자에 class, attribute, pseudo-class가 많음 그냥 요소(또는 가상 요소)가 많음

단위

      1. px 는 절대적인 값
      2. rem 은 상대적인 값(html 태그의 font-size 기준)
        2-1. 2rem 은 html 태그의 font-size의 2배 크기
      3. em 상대적인 값(자기 자신의 font-size를 기준)
        3-1. font-size를 따로 정해주지 않을 경우, 상위 요소에서 상속받은 값을 기준
      4. % 는 상대적인 값(상위 요소에 곱해주는 방식)

display

inline display

      1. inline 요소들은 다른 요소들과 같은 줄에 머무르려고 하는 성향
      2. 필요한 만큼의 가로 길이만 차지하는 성향
      3. span a b i img button
      4. display: block; 로 display block으로 바꾸기

block display

      1. block 요소들은 다른 요소들과 독단적인 줄에 가려고 하는 성향
      2. 최대한 많은 가로 길이를 차지하는 성향
      3. div h1~6 p nav ul li
      4. display: inline; 로 display inline으로 변경 가능
